## Getting Started

### 1. Install dependencies
npm install

### 2. Start dev server
npm run dev

### 3. Build for production
npm run build

### 4. Preview production build
npm run preview

## Deploy

This project is configured to deploy to GitHub Pages at `https://OSIDED.github.io/Smart-Dispatch/`.

npm run deploy
