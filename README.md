# LogisticsFlow — React App

A WhatsApp-based logistics landing page built with **React + Vite + Tailwind CSS**.

---

## Getting Started

### 1. Install dependencies
npm install

### 2. Start dev server
npm run dev

### 3. Build for production
npm run build

### 4. Preview production build
npm run preview
